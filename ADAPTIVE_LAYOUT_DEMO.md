## 📱 响应式适配

### 大屏幕 (1920px+)
- 所有图表按设计比例显示
- 最佳的视觉效果和空间利用

### 中等屏幕 (1440px)
- 5图模式调整为4列布局
- 保持良好的可读性

### 小屏幕 (1024px)
- 图表高度适当减小
- 保持核心信息可见

### 移动端 (768px以下)
- 自动切换为单列布局
- 图表堆叠显示，便于滑动查看

## 🔧 技术实现

### 核心算法
```typescript
const getResponsiveSpan = (totalCount: number) => {
  const spanMap = {
    1: { xl: 24 },      // 100% width
    2: { xl: 12 },      // 50% width each
    3: { xl: 8 },       // 33.3% width each
    4: { xl: 6 },       // 25% width each
    5: { xl: 4 },       // 20% width each (special handling)
    6: { xl: 4 },       // 16.7% width each
  };
  return spanMap[totalCount] || { xl: 4 };
};
```
